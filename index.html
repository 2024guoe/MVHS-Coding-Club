What is Coding Club?


Coding Club is a place where students can learn about coding and how to code in different coding languages while forming a creative, inclusive, and collaborative environment. For the 2022-2023 year, we plan to focus on HTML, CSS, JS, and Java. Projects will include coding and designing individual websites and tinkering with arduino board kits in groups. We will also be utilizing resources from the GWC Club Program. We welcome all students, regardless of previous coding experience, to join Coding Club! Click on the tabs to learn more about Coding Club and how to join.



The Art of Coding


Coding has positive impacts on people interested in any field. Programming helps develop problem solving skills while attempting to troubleshoot coding errors. Coding also exercises creativity, from coding every last detail into a game to aiding in website design. Furthermore, just like math and music, programming is a universal language that can be pushed out or pulled in from anywhere, but understood the same. Finally, programming cultivates community and collaboration. Whether it's participating in hackathons or joining a robotics club, coding gives an opportunity to share knowledge, skills, and jokes with like-minded people.
